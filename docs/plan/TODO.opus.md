## I. The Cognitive IDE: Making Thought Visible

*Transform complex reasoning into an observable, debuggable, steerable experience.*

### 1.1 Core Visualization System

- [ ] **Thought Graph Canvas** ‚Äî Interactive force-directed graph showing live concept activation, belief connections,
  and inference flows
- [ ] **Temporal Scrubber** ‚Äî Rewind/replay reasoning sequences to any point; "time travel" through the thought process
- [ ] **Derivation Tree Overlays** ‚Äî Expand any conclusion to see the full proof tree, color-coded by rule types
- [ ] **Attention Heatmap** ‚Äî Visualize where cognitive resources are allocated across the knowledge graph
- [ ] **Contradiction Highlighting** ‚Äî Real-time alerts when conflicting beliefs emerge, with resolution suggestions

### 1.2 Debugger Experience

- [ ] **Breakpoint System** ‚Äî Set breakpoints on specific concepts, rules, truth thresholds, or LM invocations
- [ ] **Step/Continue/Pause Controls** ‚Äî VCR-style controls with single-step, run-to-breakpoint, and batch execution
- [ ] **Watch Expressions** ‚Äî Monitor specific term relationships, truth values, or priority changes
- [ ] **Reasoning Diff View** ‚Äî Compare two reasoning sessions side-by-side to understand behavioral differences
- [ ] **Confidence Inspector** ‚Äî Click any belief to see full evidential support and truth value derivation

### 1.3 Intervention & Steering

- [ ] **Belief Surgery** ‚Äî Directly inject, modify, or suppress beliefs with full audit trail
- [ ] **Rule Toggles** ‚Äî Dynamically enable/disable specific inference rules during execution
- [ ] **Priority Override Panel** ‚Äî Manually boost or suppress concept priorities to guide attention
- [ ] **Counterfactual Sandbox** ‚Äî "What if?" branches that explore alternative reasoning paths without affecting main
  state
- [ ] **Goal Injection Interface** ‚Äî Natural language goal specification with automatic Narsese translation

### 1.4 The 10-Minute Brilliance Experience

- [ ] **One-Command Launch** ‚Äî `npm run demo` spins up entire environment with sample scenario
- [ ] **Interactive Tutorial Mode** ‚Äî `?tutorial=true` activates guided walkthrough with contextual highlights
- [ ] **"Agent Debugging" Showcase** ‚Äî Pre-built demo of catching and correcting flawed agent reasoning
- [ ] **90-Second Video Hook** ‚Äî Embedded video demonstrating core value proposition
- [ ] **Progressive Disclosure** ‚Äî UI reveals complexity only as user demonstrates readiness
