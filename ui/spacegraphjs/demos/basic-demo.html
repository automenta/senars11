Basic Demo - Recommended API Pattern

RECOMMENDED CORE LIBRARY API: SpaceGraph should implement an ergonomic async static factory method:

  • SpaceGraph.the(container, options) - should be added to core library
  • Properly handles async initialization that constructors cannot
  • Starts animation automatically for immediate visual feedback
  • Reduces boilerplate from 4+ lines to 1 line

HOW TO USE: Once implemented in the core SpaceGraph class, developers would use:

const space = await SpaceGraph.the(container);
